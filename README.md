# Todo List App

A simple and user-friendly task management application that allows users to create, view, update, and delete tasks.

## How to Run the App

- Visit https://todo-app-pgyr.onrender.com

## Technologies Used

- Frontend
    - HTML
    - CSS
    - JavaScript
    - Fetch API 

- Backend
    - Node.js
    - Express.js
    - MongoDB + Mongoose


- Deployment
    - Render

## Key Features Implemented

* Login / Sign up options
* Create new tasks
* Edit and update existing tasks
* Delete tasks
* Sort tasks by due date, pending, completed and overdue
* Persistent storage using MongoDB
* Clean and simple user interface
* Optimal display for mobile and desktop users (responsive)
